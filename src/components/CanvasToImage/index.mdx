## API

| 属性 | 说明     | 类型                                         | 默认值 |
| ---- | -------- | -------------------------------------------- | ------ |
| title | 标题 |  string必填 | 图片 |
| to | 是否转换 |   Boolean非必填 | true |
| height | 高度 |  number | 200 |
| width | 宽度 |  number | 200 |
| font | 字体设置 |  string | 'bold 53px serif' |
| style | 样式（主要设置图片背景） |  string | '' |
