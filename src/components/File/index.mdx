## API

| 属性 | 说明     | 类型                                         | 默认值 |
| ---- | -------- | -------------------------------------------- | ------ |
| data | 数据源 |  data=[{name:'文件名称',type:'file',id:1,select?:false}]必填 | [] |
| rightMenu | 右击菜单源 |  data=[{ name: string,handle(): void;]非必填 | [] |
| rightClick | 是否右击 |  Boolean非必填 | false |
